# Demo Script

## Opening

Position the platform as a compliance intelligence layer that sits above existing revenue systems and turns siloed data into explainable action.

## KRA Angle

1. Open the executive overview and explain the current revenue gap, recoverable tax, and priority risks.
2. Move to the risk queue and filter for a high-confidence VAT or import mismatch.
3. Open the taxpayer profile and show source-linked totals and relationships.
4. Create or open a case from the risk signal.
5. Generate an evidence pack and show the rule, source records, period, confidence, and recommendation.
6. Move to executive analytics and show sector, region, tax head, recovery, and officer dashboards.

## County Angle

1. Open settlement monitoring.
2. Run reconciliation or show the synthetic settlement exception list.
3. Explain missing, delayed, duplicate, and wrong-account settlement cases.
4. Open a settlement case and connect the exception to recovery follow-up.

## Commercial Close

1. Open Pilot Readiness.
2. Show ROI calculator outputs from the current dashboard data.
3. Export the pilot package PDF and ROI workbook.
4. Review procurement route, security controls, and approved-data policy.

## Guardrail

State clearly when synthetic data is being used. Do not imply that demo figures are real taxpayer findings.
