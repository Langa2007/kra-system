# Phase 16 Pilot Package

This folder is the commercial-readiness pack for controlled KRA, county, integrator, or public finance reform conversations.

## Contents

- `pilot-proposal.md`: pilot scope, stakeholders, workplan, and success measures.
- `demo-script.md`: live demo sequence covering KRA and county revenue angles.
- `security-overview.md`: identity, access control, audit, privacy, and export controls.
- `data-processing-overview.md`: synthetic-first demo policy and approved pilot data handling.
- `deployment-overview.md`: local demo and controlled pilot deployment stack.
- `sample-evidence-packs.md`: sample evidence-pack structure and demo examples.
- `sample-dashboards.md`: dashboard inventory and buyer questions each screen answers.
- `pricing-procurement.md`: pricing model and procurement route notes.

## Demo Rule

Use synthetic data unless a named pilot owner has approved the data source, purpose, users, and retention period.
